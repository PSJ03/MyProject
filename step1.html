Step 1: HTML5와 보안 기초

안전한 웹의 시작, 시맨틱 구조 이해

HTML5 정의

웹 문서를 구성하는 가장 기본 언어로, 하이퍼텍스트를 통해 정보를 구조화합니다.

시맨틱 태그

의미 있는 태그를 사용하여 검색 엔진 최적화(SEO)와 웹 접근성을 향상시킵니다.

구조와 보안

올바른 마크업은 비정상적인 스크립트 삽입을 방어하는 기초 보안 설계가 됩니다.

웹 개발 필수 태그 정보

사용자 데이터를 서버로 전달하기 위한 핵심 구성 요소입니다.

  • 입력 태그: <form>, <input>, <select>
  • 전송 버튼: <button type="submit">

정보를 시각적으로 구조화하고 미디어를 배치하는 방법입니다.

  • 미디어: <img>, <video>, <audio>
  • 테이블: <table>, <tr>, <th>, <td>
