Step 3: JavaScript - 동적 웹의 완성

사용자 경험을 개선하는 프로그래밍 언어의 역할

자바스크립트의 역할

  • 웹 요소의 스타일과 내용을 실시간으로 제어
  • 사용자의 클릭, 입력 등 이벤트 처리
  • 서버와의 비동기 통신을 통한 데이터 갱신
※추후 자바스크립트를 활용하여 정보보호 퀴즈 및 진도 체크 기능을 구현할 예정입니다※
DOM (Document Object Model)

브라우저는 HTML 문서를 계층 구조를 가진 객체 모델(DOM)로 생성합니다. 자바스크립트는 이 DOM 트리에 접근하여 요소를 추가, 삭제, 수정함으로써 동적인 화면을 구성합니다. 

document.querySelector('#id').style.color = 'blue';
