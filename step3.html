Step 3: JavaScript와 동적 웹

사용자 경험 개선과 동적 상호작용

Javascript: 동적 상호작용

자바스크립트는 정적인 HTML/CSS에 생동감을 불어넣는 언어입니다. 사용자의 동작에 실시간으로 반응하고 데이터를 처리하여 웹의 완성도를 높입니다.

요소 제어: 실시간 스타일 및 텍스트 변경
이벤트 처리: 클릭, 마우스 오버 등 사용자 동작 감지
DOM 제어: 웹 문서 구조를 객체로 다루는 기술
interaction.js 
btn.addEventListener('click', () => {
  document.body.style.color = 'blue';
  alert('이벤트가 발생했습니다!');
});

안내: 현재 페이지는 HTML/CSS 구조 전시용입니다. 실제 동적 기능과 퀴즈 시스템은 기말고사 프로젝트에서 구현될 예정입니다.
