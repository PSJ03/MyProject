Step 2: CSS3 - 스타일링과 레이아웃

다양한 환경에 대응하는 반응형 웹 디자인 기술

CSS 박스 모델 (Box Model)

모든 HTML 요소는 사각형 박스로 취급됩니다. 콘텐츠(Content), 안쪽 여백(Padding), 테두리(Border), 바깥 여백(Margin)의 유기적인 관계를 이해하는 것이 레이아웃의 기초입니다.

반응형 웹 디자인 (Responsive)

미디어 쿼리(Media Queries)를 사용하여 사용자의 기기 환경에 맞는 최적의 레이아웃을 제공합니다. 모바일 우선(Mobile-First) 전략은 현대 웹 개발의 표준입니다.

Bootstrap 프레임워크

트위터에서 개발한 오픈소스 CSS 프레임워크로, 복잡한 CSS 작성 없이도 강력한 그리드 시스템과 컴포넌트를 활용하여 일관성 있는 반응형 웹을 빠르게 구축할 수 있게 도와줍니다.
